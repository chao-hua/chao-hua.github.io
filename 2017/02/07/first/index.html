Sublime 结合 Evernote 保存 Markdown

Evernote 不支持 Markdown 一直是个缺陷.

Sublime 的 Evernote 插件，就是来解决这个问题的。
在 Sublime 中显示的是 Markdown 语法，在 Evernote 中显示的是格式化后的样式。
支持双向编辑，但是在 Evernote 中编辑后，在 Sublime 中会出现标签，这样会惨不忍睹，
所以应当避免在 Evernote 中编辑 Markdown 笔记（似乎又变成的单项编辑…）。

安装 Sublime 的 Evernote 插件

获取 Evernote 应用授权

获得 Developer Token 和 NoteStore URL

点击网址 印象笔记(中国) 或者 Evernote 登录后来获取信息。
在 Sublime 中配置授权信息，Preferences > Package Settings > Evernote > Settings - User 中添加

配置 Evernote 快捷键

可以参照 官网 进行配置
我自己的配置如下，在 Preferences > Key Bindings > User 中添加

  • 使用 Ctrl+Shift+E 直接调用与 Evernote 相关的 Menu。不常用的此插件功能可以通过此快捷键调出 Menu 后查询使用。
  • 加入了新建笔记快捷键（Ctrl+E, Ctrl+N 连续按键，其他绑定两个键的使用方式相同）
